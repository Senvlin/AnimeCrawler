这是一款基于 <code>[**Ruia**](https://github.com/howie6879/ruia)</code> 轻量级框架、在Windows平台下专门爬取免费动漫的爬虫，底层使用<code>aiohttp</code>库，使用异步模式，大幅增加爬取及下载速度。可**离线播放**各大动漫，立志成为最实用，最轻量的动漫管理及下载助手
